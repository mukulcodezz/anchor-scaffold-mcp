I shipped my first developer tool: **Anchor Scaffold MCP** 🚀

Building on Solana with the Anchor framework means writing the same boilerplate over and over — account structs, PDA derivations, space calculations, TypeScript clients, test suites. Hours gone before you even touch your actual logic.

So I built a tool that generates it.

Feed it your Anchor IDL (or just describe what you want), and it produces production-ready:

✅ TypeScript clients — typed instruction callers, PDA helpers, account fetchers
✅ Rust #[derive(Accounts)] structs — every constraint, correct space math
✅ Mocha/Chai test suites
✅ Complete Anchor programs from a plain-English description

It runs two ways:
→ As an MCP server inside Claude Code ("generate a client for my program")
→ As a standalone CLI

Other details I cared about:
• Multi-backend — Claude, OpenAI, or any OpenAI-compatible endpoint
• Real prompt caching → ~90% cheaper on repeated generations
• Path-traversal guards, input validation, 19 unit tests, CI on every push
• Open source, MIT

🔗 Live site: https://mukulcodezz.github.io/anchor-scaffold-mcp-website/
💻 GitHub: https://github.com/mukulcodezz/anchor-scaffold-mcp
📦 npm: npm install -g anchor-scaffold-mcp

If you build on Solana, I'd love feedback — and if you've got a feature in mind, open an issue.
